# DataConverter

A Python module to convert data between CSV, JSON, and XML formats. This module provides functions to convert data in both directions for each format.

## Features

- Convert CSV to JSON
- Convert CSV to XML
- Convert JSON to CSV
- Convert JSON to XML
- Convert XML to CSV
- Convert XML to JSON
